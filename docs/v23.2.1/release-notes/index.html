Release Notes

NOTE

This version of the Skillz SDK supports Xcode 11 and iOS 13. To integrate v23.2.1 through Xcode, developers must use Xcode 11.

This release has new features and includes quality of life improvements for Unity.

Changes

Increased Android Minimum, Target, and Compile API Versions

The Skillz SDK no longer supports Android 4.4, 5.0, or 5.1 (API Level 19 through 22). The minSdkVersion required for a Skillz Android project is now API 23 (Android 6.0). Additionally, the targetSdkVersion and compileSdkVersion is now API 28 (Android 9.0). Please make sure to check the Upgrade Notes for details.

Added Android 64-bit Support

The SDK now has direct support for compiling to 64-bit Android devices. Refer to the Upgrade Notes for more information.

Turn-Based Matches Are No Longer Available

Skillz no longer offers turn-based matches. As a result, the turn-based APIs have been removed from version 23 of the Skillz iOS SDK. Please update your game to remove references to these APIs.

Note: This refers to asynchronous turn-based match types. Realtime turn-based match types are not affected.

New Features

Unity iOS

Developers will no longer need to check the "Code Sign on Copy" setting in the exported Xcode project. This will make iterating on Unity iOS a little quicker. You can find our complete Unity SDK Installation guide here

Unity Android

The Skillz SDK no longer supports building an APK directly from Unity. Please export your project to Android Studio when you need to build an APK. Refer to the instructions for detailed steps on how to do this.

OnSkillzWillExit() fix

This release fixes a bug where backgrounding your app, while the Skillz UI was displayed, then foregrounding it would inadvertantly cause the OnSkillzWillExit() method of the SkillzMatchDelegate to be called.

Updated Store

Players now have new access points to check their Ticketz and Trophy status within the Store. By adding a "Trophies and Ticketz" section to the Store, players view both of these loyalty features more often. With more reasons for players to visit the Store, there is an overall increase in visits to this critical monetization screen.

Login Bonus

Players can now receive Ticketz, Bonus Cash or Virtual Currency for simply logging into a game, visiting the Store and clicking the "Login Bonus" button. All of the rewards are customizable by player segment; meaning players have unique Trophy experiences that match their gaming behavior. Both of which are great for player retention.

Since the Login Bonus is located on the Store, more players are checking out Limited Time Offers and Bonus Cash Offers. This will result in an increasing amount of players depositing funds into the Store.

Android Updates

There have been multiple UI and UX Android updates, leading to a smoother and snappier experience with fewer user crashes. With added behaviors to the Android Back Button, players are more familiar with the UX of your game.

More Features Coming Soon

There are tons of new and exciting features that are coming soon to v23. Stay tuned!
