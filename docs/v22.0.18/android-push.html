Android Push Notifications

This article will walk you through setting up push notifications for your Android based game using Google's Firebase.

Set Up a Firebase Project

Please follow Google's instructions on Add Firebase to your Android Project. Follow along with Option 1: Add Firebase using the Firebase console

Note: If you run into any runtime errors, you may need to use firebase-core version 11.2.2.

Upload the API key to Skillz

Go to https://console.firebase.google.com/, and select your project. Select Settings (top left corner of the console)

  1. Select Cloud messaging tab.
  2. Copy the server key.
  3. Go to your Skillz developers console and select your application.
  4. Select the Push Notification tab on your left.
  5. In the Android - GCM Push notification, paste your server key to the blank text field, and hit Save:

That's it, push notifications are now enabled for your Android game.
