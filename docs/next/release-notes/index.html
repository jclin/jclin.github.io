Release Notes

NOTE

This version of the Skillz SDK supports Xcode 11 and iOS 13. To integrate v23.2.1 through Xcode, developers must use Xcode 11.

This release has new features and includes quality of life improvements for Unity.

Changes

Increased Android Minimum, Target, and Compile API Versions

The Skillz SDK no longer supports Android 4.4, 5.0, or 5.1 (API Level 19 through 22). The minSdkVersion required for a Skillz Android project is now API 23 (Android 6.0). Additionally, the targetSdkVersion and compileSdkVersion is now API 28 (Android 9.0). Please make sure to check the Upgrade Notes for details.

Added Android 64-bit Support

The SDK now has direct support for compiling to 64-bit Android devices. Refer to the Upgrade Notes for more information.

AndroidX Required

The AndroidX Support Libraries are required by the Skillz SDK instead of the original Android Support Libraries. Refer to the "Enable AndroidX" section in the Upgrade Notes on how to migrate your Android Studio project to use the new libraries.

Turn-Based Matches Are No Longer Available

Skillz no longer offers turn-based matches. As a result, the turn-based APIs have been removed from version 23 of the Skillz iOS SDK. Please update your game to remove references to these APIs.

Note: This refers to asynchronous turn-based match types. Realtime turn-based match types are not affected.

New Features

Unity SDK

This release introduces new and exciting features to the Unity SDK.

Unity Companion

Prior to v25.x, testing a game's workflow with Skillz involved a bit of overheard. Developers had to export to either Android Studio or Xcode, and then compile and run the game. This was often a time consuming process. With the Unity Companion, testing a game's workflow with Skillz can be done entirely in the Unity editor. When a game is run in the editor, the Skillz SDK will display low-fidelity screens to simulate a basic game workflow with Skillz. Refer here for further reading.

Skillz Settings

The Skillz Unity SDK now has a dedicated settings window. This consolidates configuration of your game for Skillz in a single location, and removes the need for a special SkillzDelegate game object.

Google Play Services Resolver Support

v25.x bundles the Google Play Services Resolver plugin. This is a popular plugin that resolves Android dependencies between multiple libraries. Integrating Skillz with your Unity based game for Android has now become even easier. Refer to either the upgrade or fresh installation instructions for more details.
