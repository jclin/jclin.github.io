Sandbox Payment Credentials

In order to test cash games under the Skillz Sandbox environment you may need to make a deposit. Using the test credentials described below you can successfully make deposits without giving out real payment information.

Credit Card

A test credit card deposit can made using the follow information:

Card Number: 4242 4242 4242 4242
Expiration: 12/30
CVV: 333
Zip: 33333

PayPal

These sandbox PayPal credentials can be used to create a test PayPal deposit:

Username: paypal-test@skillz.com
Password: qweasdzxc

Apple Pay

When testing Apple Pay, be sure that you are on a device that supports Apple Pay, you have Apple Pay setup on your device, and the credit card on file is supported by Skillz. We do not show the Apple Pay button to players in game if they do not meet these conditions.

NOTE

You will need to test Apple Pay using a real credit card, and not the sandbox credit card listed above. But rest assured, as long as the game is in Sandbox environment, your credit card will not be charged!
